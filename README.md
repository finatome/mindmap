mindmap
  root((Black-Scholes Theorem))
    "1. Core Idea"
      "Goal: Fair price of a European option (no arbitrage)"
      "Key Formula: V = S * N(d1) - K * exp(-r*(T-t)) * N(d2)"
      "Put via Put-Call Parity"
    "2. Foundations"
      "Underlying Dynamics: dS = mu*S*dt + sigma*S*dW"
      "Market Assumptions"
        "Frictionless market (no costs/taxes)"
        "Constant r, sigma"
        "Continuous trading"
        "No arbitrage"
        "Borrow/lend at risk-free rate"
        "No dividends"
      "Instruments"
        "Stock S_t"
        "Bond B_t = exp(r*t)"
    "3. Building Blocks"
      "Replicating Portfolio"
        "Option = Delta shares + B bond"
        "Choose Delta to eliminate risk (cancel dW term)"
      "Self-Financing Condition"
        "Portfolio value changes only by market movement"
        "dV = Delta*dS + r*(B - Delta*S)*dt"
      "No-Arbitrage Argument"
        "Riskless portfolio must earn rate r"
        "Leads to PDE: V_t + 0.5*sigma^2*S^2*V_SS + r*S*V_S - r*V = 0"
    "4. PDE Solution"
      "Boundary: V(S,T) = max(S - K, 0)"
      "Define:"
        "d1 = [ln(S0/K) + (r + 0.5*sigma^2)*T] / (sigma*sqrt(T))"
        "d2 = d1 - sigma*sqrt(T)"
      "Solutions:"
        "Call: C = S0*N(d1) - K*exp(-r*T)*N(d2)"
        "Put:  P = K*exp(-r*T)*N(-d2) - S0*N(-d1)"
    "5. Outputs"
      "Option Values (Call/Put)"
      "Greeks: Delta, Gamma, Theta, Vega, Rho"
      "Hedging Ratios and Risk Measures"
    "6. Interpretations"
      "Delta Hedging: continuous risk elimination"
      "Risk-Neutral Valuation: replace mu with r"
      "Volatility drives time value"
      "Foundation of modern derivatives pricing"
    "7. Extensions"
      "Dividends: replace S0 with S0*exp(-q*T)"
      "American Options: free-boundary problems"
      "Stochastic Volatility: Heston model"
      "Jump Diffusion: Merton model"
      "Binomial Model: discrete-time precursor"
